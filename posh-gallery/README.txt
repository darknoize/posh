POSHMFG interactive product gallery

Files included:
- index.html
- styles.css
- script.js
- assets/ (images)

How it works:
- Main stage image updates with previous/next arrows.
- Thumbnails below the main image are clickable.
- Clicking a thumbnail updates the active large slide.
- Detail panel updates at the same time with a larger product image on the left and text on the right.

Edit product copy in script.js.
Swap any image references in the products array if you want to use different assets.
